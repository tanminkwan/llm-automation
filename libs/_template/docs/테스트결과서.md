## 9. 회귀 / 기지 결함

검증 중 발견되어 해결된 이슈:

1. **uv 패키지명 규칙 위배** — `name = "_template"` 은 uv 에서 reject ("must start with letter/digit"). → `seed-template` (배포명) 으로 변경, import 명은 `_template` 유지.
2. **ruff per-file-ignores 패턴 miss** — `tests/**/*.py` 는 workspace 중첩 경로(`libs/_template/tests/`)에 매칭 안 됨 → `**/tests/**/*.py` 로 수정, `**/conftest.py` 추가.
3. **Docker 태그 밑줄 금지** — Makefile `SVC=_template` 을 그대로 tag 로 쓰면 실패 → `IMAGE := seed-template` 추가하여 분리.
4. **tester stage coverage 0%** — wheel 설치 후 `--cov=src/_template` 측정 시 site-packages 의 코드가 측정 대상 밖이라 0% → `pip install -e .` (editable) 로 수정.
