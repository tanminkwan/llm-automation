## 9. 회귀 / 기지 결함

검증 중 발견되어 해결된 이슈:

1. **`tests/test_factory.py` 에서 relative import 실패** — `tests/` 가 패키지가 아니어서 `.conftest` import 불가. 해결: `tests/__init__.py` 추가.
2. **ruff lint 미사용 import** — `test_factory.py` 에 `Any`, `AgentResult`, `ToolSpec` 미사용 import 존재. 해결: 제거.
3. **ruff format 미일치** — 4개 테스트 파일 포맷 불일치. 해결: `make format` 실행.
4. **`S108` 경고** — `test_settings.py` 에서 `/tmp/agent-runner` 리터럴 비교. 해결: `noqa: S108` 주석 추가 (설정값 검증이므로).
