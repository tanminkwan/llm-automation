## 7. 실행 중 발견 및 해결한 이슈

1. **uv 패키지명 규칙** — `libs/_template/pyproject.toml` 의 `name = "_template"` 을 uv 가 reject. `seed-template` 으로 변경. import 명(`_template`)은 `packages = ["src/_template"]` 경로 지정으로 영향 없음.
2. **ruff per-file-ignores 경로 매칭** — 루트에 정의한 `tests/**/*.py` 패턴이 workspace 중첩 경로(`libs/_template/tests/`)에 매칭 안 됨. `**/tests/**/*.py` + `**/conftest.py` 로 보정.
3. **Docker 이미지 태그 밑줄 금지** — `_template:latest` 빌드 실패. 멤버 Makefile 에 `IMAGE := seed-template` 를 분리 추가 (`SVC=_template` 은 로깅/식별 용도 유지).
4. **tester stage 커버리지 0%** — wheel 설치 후 `src/` 경로 측정 불가. `pip install -e .` (editable) 로 변경하여 해결.

본 4건은 `libs/_template/docs/테스트결과서.md §9` 에도 기록.
