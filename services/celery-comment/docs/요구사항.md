## 1. 개요

`comment_queue` 에서 태스크를 consume 하여 AI Agent CLI(AgentRunner)를 실행하고, RAG MCP 도구를 활용해 Java 코드에 주석을 삽입한 뒤 Git commit/push 하는 Celery 워커 서비스.

## 2. 기능 요구사항 (FR)

| ID | 기능 | 설명 |
|---|---|---|
| FR-01 | Celery task 정의 | `comment.process` task — `comment_queue` 에서 consume |
| FR-02 | Git clone | `clone_url` 로 작업 디렉터리에 repo clone |
| FR-03 | AgentRunner 실행 | AgentRunner.run() 호출 — RAG MCP tool 등록, 프롬프트 전달 |
| FR-04 | 결과 처리 | AgentResult 에서 변경 파일 확인 → Git add/commit/push |
| FR-05 | 실패 처리 | AgentRunner 실패 시 에러 로깅 + task FAILURE 상태 |
| FR-06 | 작업 디렉터리 정리 | task 완료 후 임시 디렉터리 삭제 |
